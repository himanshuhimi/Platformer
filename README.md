### Platformer | Bunny Adventure

A game about a small bunny jumping and collecting fruits!! Play to see more!

**NOTE:**
1. This is the source code of the game.
2. To run the game, You'll need the following DLL Files on your system:
- libgcc_s_seh-1
- libstdc++-6
- libwinpthread-1
- SDL3
- SDL3_image
- SDL3_ttf
- SDL3_mixer
